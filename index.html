Introduction

The data is from mtcars dataset which details the different attributes of cars, including cars of different gear. We would want to see how different gear changes the average MPG. This application allows users to change the no of gear and immediately see the average MPG when submitted.

Components

  1. Documentation of App use
  2. Entry of No of Gear
  3. Results of average MPG

Documentation

This describe the App and what we are trying to achieve.

Entry of No of Gear

The range of values are 3 to 5. User can vary the value and see the results upon submit

Results

This gives the average MPG of the selected gear.
